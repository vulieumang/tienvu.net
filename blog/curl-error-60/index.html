Fix cURL error 60: SSL certificate

15/12/2025 | WordPress Lexicon

Nhiều bạn gặp lỗi cURL error 60: SSL certificate unable to get local issuer certificate khi làm việc với WordPress trên localhost. Đây là cách fix nhanh nhất.

Giải pháp

Chỉnh sửa file functions.php của theme đang kích hoạt (Appearance > Theme Editor).

Thêm dòng sau vào cuối file:

add_filter('https_ssl_verify', '__return_false');

Đoạn mã này sẽ tắt việc xác thực SSL cho các request HTTP, giúp bỏ qua lỗi chứng chỉ trên môi trường local.

Lưu ý: Chỉ nên sử dụng giải pháp này trên môi trường Localhost (phát triển). Không nên dùng trên môi trường Production (website thực tế) vì lý do bảo mật.
