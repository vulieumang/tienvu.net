Cách cài đặt file .msix trên Windows 10 LTSC

15/12/2025 | Windows Tips

Một số máy Windows 10 LTSC sẽ không có Windows Store nên không thể cài đặt file .msix theo cách thông thường. Dưới đây là hướng dẫn cài đặt bằng PowerShell.

1. Cài đặt môi trường cần thiết

Mở PowerShell và chạy lệnh sau:

Add-AppxPackage 'https://aka.ms/Microsoft.VCLibs.x64.14.00.Desktop.appx'

Nếu gặp lỗi Deployment failed with HRESULT: 0x80073CF3 hoặc thiếu Microsoft.VCLibs, bước này sẽ khắc phục vấn đề.

2. Cài đặt file .msix của bạn

Sử dụng lệnh sau, thay thế đường dẫn đến file của bạn:

Add-AppxPackage -Path "C:\duong-dan-den-file\yourfile.msix"

Hoặc cài trực tiếp từ URL (Ví dụ TradingView):

Add-AppxPackage -Path "https://tvd-packages.tradingview.com/stable/latest/win32/TradingView.msix"
Dành cho người dùng TradingView: Chạy lần lượt 2 lệnh sau để cài đặt nhanh: 
Add-AppxPackage 'https://aka.ms/Microsoft.VCLibs.x64.14.00.Desktop.appx'
Add-AppxPackage -Path "https://tvd-packages.tradingview.com/stable/latest/win32/TradingView.msix"
